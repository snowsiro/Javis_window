## 핵심 결정: "대본 + 녹음 + 파형"을 어떻게 같이 보여줄까

파형 UI가 **실제 TTS 음성에 실시간 반응**하므로, 가장 강력한 방법은:

### ✅ 추천 (Option A) — 라이브 캡처
자비스 대사를 **실제 앱의 TTS로 재생**하면서 그 순간 **JARVIS UI 화면을 화면 녹화**.
→ 파티클이 자비스 목소리에 맞춰 튀는 게 자동으로 싱크됨(이미 그렇게 동작). 편집에서 따로 맞출 필요 없음.
- 장점: 싱크 완벽, "진짜 작동하는 자비스" 증거. 메타가 강력.
- 촬영: 주인이 카메라 정면(스토리) ↔ 자비스 말할 때 JARVIS UI 풀화면(파티클 반응)으로 전환.
- 인스타식 "뒤통수 + UI" 샷은 데모 구간에 OK, 인트로/클로징은 얼굴 정면.

### 대안 (Option B) — 분리 후 편집 싱크
자비스 음성 파일을 미리 뽑고, UI는 따로 캡처해 편집에서 맞춤. 비추천(번거롭고 싱크 깨짐).
